# AusHello

> AusHello is location-based app to connect refugees to find critical local resources in communities around Australia.

## Stack
- Back end: Ruby on Rails
- Database: PostgreSQL
- Front end: HTML5, CSS3, JavaScript (ES6), Bootstrap
- Hosting: Heroku
